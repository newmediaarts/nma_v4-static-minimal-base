Kapi‘olani Community College

New Media Arts

The New Media Arts (NMA) program is a two year AS degree program at Kapi‘olani Community College in Honolulu, Hawaii. Our mission is to prepare students for employment in the fields of interface design and animation.

Study

Animation

The Animation specialization prepares students for careers in 3D computer animation, video game development, and emerging industries employing 3D computer graphics. Topics include the full computer graphics (cg) pipeline, film analysis and storytelling, digital painting, and demo reel development.

Learn About the NMA Animation Curriculum

Study

Interface Design

The Interface Design specialization prepares students for careers as interface designers with a focus on web design. Topics include graphic design, typography, graphic symbolism, user interface design, front-end web development, and motion graphic design. Additional courses include digital imaging, design history, and portfolio development.
